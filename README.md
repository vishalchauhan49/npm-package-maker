# A `npm package` which will do simple mathematical operations.  

How To Install Initialise In Your Project 

> **NOTE:This npm package is accessible in ESM module.**



```
 npm i cal-math 
      OR 
 npm install cal-math

import { math } from "cal-math"
```


> It provides some functions and they are :-- math() is main function
- `math.add(num,num)`:- Returns addition of two number 
- `math.sub(num,num)`:- Returns subtraction of two number 
- `math.mul(num,num)`:- Returns multiplication of two number 
- `math.div(num,num)`:- Returns division of two number
